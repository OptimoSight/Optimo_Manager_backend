This is how easy it is to add virtual try-on to your e-commerce site:

1. Include the Widget Files

<!-- In your HTML head --> <link rel="stylesheet" href="https://localhost:42413/static/css/vto-widget.css"> <!-- Before closing body tag --> <script src="https://localhost:42413/static/js/vto-widget.js"></script>

2. Add the Try Now Button

<button class="vto-try-now-btn" data-category="lipstick" data-color="#e91e63" data-color-name="Rose Pink" data-product-name="Your Product Name" data-api-key="OptimosightGuest999" data-mode="both" data-colors="#dc2626,#e91e63,#7c2d12,#991b1b" data-color-names="Ruby Red,Rose Pink,Chocolate,Deep Red"> ‚ú® Try Now </button>

3. Dynamic Color Selection (JavaScript)

// Update button when color changes function updateTryOnColor(button, color, colorName) { button.setAttribute('data-color', color); button.setAttribute('data-color-name', colorName); }
